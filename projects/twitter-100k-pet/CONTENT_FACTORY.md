## 两个生产流程（Daily）

### Flow-A：实时/热点快反流（速度优先）
适用：当天新闻、平台热梗、行业热点。

1. 选题输入（每天 3-5 条）
   - 来源：热搜、行业新闻、社交热议
   - 筛选标准：高讨论度 + 可两格表达 + 有立场/反差
2. 角度设计
   - 角度模板：爆笑 / 警示 / 教育 / 深思 / 创造力
3. 产出
   - 每条热点生成 2-3 个版本（不同角度）
4. 质检
   - 文本是否可读
   - 反转是否成立
   - 是否有误导/事实风险
5. 入队
   - 通过后进入发布任务卡，按 09:00 / 13:00 / 21:00 节奏发布

目标：快、准、可传播（当天打穿）

### Flow-B：生活/常青叙事流（稳定优先）
适用：有趣生活小事、职场场景、学习拖延、情绪共鸣。

1. 选题输入（每天 5-10 条）
   - 来源：真实生活观察、用户评论、历史高互动话题
2. 分类打标
   - 爆笑 / 警示 / 教育 / 引人深思 / 创造力
3. 产出
   - 每个分类至少 1 条，保证内容维度覆盖
4. 复盘
   - 曝光、互动率、转粉、发布时间偏差
5. 模板升级
   - 每周保留 Top10 模板，淘汰低效模板

目标：稳、持续、可规模化（周级复利）

## 多分类内容矩阵（执行版）

1) 爆笑（高转发）
- 机制：强反差 + 日常共鸣
- 句式：立flag -> 秒打脸

2) 警示（高讨论）
- 机制：错误决策 -> 后果反转
- 句式：以为省事 -> 结果更亏

3) 教育（高收藏）
- 机制：常见误区 -> 正确做法
- 句式：错法一句 + 正法一句

4) 引人深思（高评论）
- 机制：价值冲突 -> 留白提问
- 句式：第一格现实，第二格反问

5) 创造力（人设加分）
- 机制：脑洞设定 -> 意外落地
- 句式：夸张设定 + 有逻辑收口

---

## 每日最小交付（MVD）
- 热点快反：3 条（每条至少1张）
- 生活常青：5 条（分类覆盖）
- 最终入发布队列：>= 6 条
- 每日复盘记录：曝光/互动/转粉 + 计划/实际发布时间偏差

## 每周节奏
- 周一到周五：生产 + 发布 + 轻复盘
- 周六：汇总周报 + 模板排名 + 下周选题池
- 周日：只做高质量样本与策略迭代

## 下步可自动化
- 热点抓取 -> 角度建议 -> 批量生成脚本
- 自动将“通过内容”写入发布任务卡
- 自动生成周复盘摘要与下周选题建议
