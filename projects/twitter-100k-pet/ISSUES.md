## 待优化项（非阻塞）

1. 前端构建体积偏大
   - 现象：Vite 构建提示 chunk > 500KB（主包约 2.4MB，gzip 约 760KB）
   - 影响：首屏加载与弱网体验
   - 建议：
     - 使用 dynamic import 拆分功能模块（复盘面板、预览区、导出模块）
     - 配置 manualChunks
     - 延迟加载重量级依赖（如 html2pdf）

2. 指标数据存储仅本地 localStorage
   - 现象：当前复盘数据仅在本浏览器保存
   - 影响：跨设备不同步、清缓存会丢失
   - 建议：
     - 增加 JSON 导出/导入
     - 后续接简单后端存储（例如 sqlite/api）
