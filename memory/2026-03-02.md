## 漫画生图链路验证（新增）

- 用户要求：按“漫画提示词”生成 5-10 张样图并查看文本效果。
- Grok链路实测结论：
  - 使用版权角色直指提示时，出现上游拦截/失败（502 upstream_error）。
  - 改为“原创角色 + 粗糙蜡笔手绘风”后可生成（7/8 成功）。
  - 观察到常见问题：图中对话框可能混入日语或乱码（模型内嵌文字不稳定）。
- A方案（先无字生图，再中文覆字）已执行：
  - 路径：`projects/twitter-100k-pet/generated-comics/2026-03-02/planA/`
  - 成功 7/8；已输出 `comic_cn_*.jpg`。
- 用户提供 Google API Key 后，切换 Google 生图测试：

  - 批量结果：8/8 成功
